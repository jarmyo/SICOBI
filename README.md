# Sistema de control de Bienes

Software para control de bienes e inventario.
Permite tener control sobre bienes (PCs, mobiliarios, vehiculos) en una institución o corpotativo.

## Caracteristicas

- Impresión automatica de hojas de resguardo, etiquetas y documentos.
- Busqueda y clasificación de bienes
- Monitoreo en tiempo real de equipos informaticos (PCs, inpresoras, Telefonos, Servidores).
- Asignación de usuarios, formatos de resguardo automatizados.
- Importación desde EXCEL

## Modulos.

- Servidor Web
- Servicio Windows de monitoreo por PC
- App mobil de scaneo.

## Requerimientos.

- SQL server.
- .NET 6
- Dirección estatica accesible
